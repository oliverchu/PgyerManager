# PgyerManager/蒲公英管理器
用蒲公英开放API做的一个上传应用PC程序，您只需要打包完成后，通过Studio的Terminal输入```pgyer -c [您自定义的命令]``` 即可上传到您的蒲公英上面

链接和您自定义的模板自动复制到剪贴板，解放双手

您也可以打开程序主页面进行上传配置等等
